자주 묻는 질문

요금제
처음 가입하면 무료인가요?
로그인 후 30일간 프리미엄을 무료로 체험할 수 있습니다. 체험 기간 종료 후 별도로 구독 결제를 하지 않으면 자동으로 유료 전환되지 않습니다.
알림
공시 알림이 오지 않아요.
기기 설정 → 앱 → 다트워치 → 알림에서 알림이 허용되어 있는지 확인해 주세요. 또한 구독 탭에서 원하는 종목과 공시 유형이 구독 설정되어 있어야 알림을 받을 수 있습니다.
알림이 늦게 오는 것 같아요.
공시는 DART 서버에 등록된 후 전달되므로, 서버 사정 또는 네트워크 상태에 따라 짧은 지연이 발생할 수 있습니다.
어떤 시장의 공시를 알려주나요?
코스피(유가증권시장)와 코스닥 상장 종목의 공시를 모두 제공합니다. 구독 탭에서 코스피/코스닥 전체 시장 알림을 설정하거나, 개별 종목을 추가해 관심 종목만 받을 수 있습니다.
검색
검색으로 어디까지 찾을 수 있나요?
무료 사용자는 오늘 공시만, 프리미엄 사용자는 최근 1년치 공시를 검색할 수 있습니다. 회사명 또는 종목코드로 검색 후, 공백을 입력하면 공시 유형(분기보고서, 유상증자 등)을 추가로 필터링할 수 있습니다.
검색 예시가 있나요?
삼성전자 → 삼성전자 전체 공시 / 삼성전자 분기보고서 → 분기보고서만 / 005930 유상증자 → 종목코드로 검색 후 유상증자만 필터
플랜
무료와 프리미엄의 차이는 무엇인가요?
무료: 관심 종목 최대 3개, 오늘 공시만 검색, DART 사이트 외부 열기 / 프리미엄: 관심 종목 무제한, 최근 1년 공시 검색, 앱 내 네이티브 뷰어(모바일 최적화), 전체 공시유형 알림
30일 무료 체험이란 무엇인가요?
신규 가입 시 프리미엄 기능을 30일간 무료로 이용할 수 있습니다. 체험 기간 종료 후에는 자동으로 무료 플랜으로 전환됩니다.
기타
공시 내용이 실제와 다를 수 있나요?
다트워치는 금융감독원 DART API를 통해 공시를 수신합니다. 원본 데이터는 DART 공식 사이트(dart.fss.or.kr)를 기준으로 하며, 중요한 투자 판단 전에는 반드시 원문을 직접 확인하시기 바랍니다.
문의는 어떻게 하나요?
앱 내 내 정보 탭의 문의하기를 통해 이메일로 문의해 주세요. 빠른 시일 내에 답변드리겠습니다.
